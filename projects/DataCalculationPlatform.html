Abstract (ABSTRACT):
1.Idea, Feature
The effective classification, application, visualization, and preservation of hydrological data have become the primary goals of many huge data systems. In Taiwan, the average water consumption is insufficient due to the uneven distribution of rainfall in winter and summer, the large number of barren streams and rivers, the steep slopes and rapid flows, and the small population. Therefore, some residents use groundwater for aquaculture and agriculture, but it also causes the problem of ground subsidence. The unpredictability of the data due to the impact of the natural environment also makes the amount of data important. This project is based on the Internet of Things and big data monitoring to develop an Internet of Things implementation platform for water resources, which interfaced with existing groundwater observation website data and unified the data management design to refine effective classification, application, and display, providing better user experience and more accurate strategic planning for groundwater management and application personnel in industry, government, and academia.
2. Method
The system uses a virtual machine to set up a web site for users to use, using front-end technology to provide user interface, map, layer display and other functions, while the back-end uses an independent relational database to manage accounts, non-relational database to manage layer data, and a time series database to manage water level timing data.
3. Simple Conclusions
Therefore, this project designs and develops the hydrological and stratigraphic subsidence information system using big data as the main data, and combines the functions such as visualization of time series data, display of stratigraphic data, environmental analysis, stratigraphic data analysis, account backend management, etc.
